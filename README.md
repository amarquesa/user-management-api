# User Management API

REST API developed with Spring Boot for user management.

## Technologies

- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## Endpoints

GET /users  
POST /users  
PUT /users/{id}  
DELETE /users/{id}

## Example Request

POST /users

{
"name": "Amanda",
"email": "amanda@email.com"
}
